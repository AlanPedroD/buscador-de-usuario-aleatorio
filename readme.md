# 🎲 Gerador de Usuário Aleatório

Aplicação web simples que consome uma API externa para gerar e exibir usuários aleatórios em tempo real.

---

## ✨ Funcionalidades

- Geração de usuários aleatórios com um clique
- Consumo de API com `fetch` e `async/await`
- Exibição dinâmica de nome e avatar
- Feedback de carregamento para melhor UX
- Tratamento de erros na requisição
- Botão desabilitado durante o carregamento

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |
|---|---|
| HTML5 | Estrutura da aplicação |
| CSS3 | Estilização e layout |
| JavaScript (Vanilla JS) | Lógica e interatividade |
| [ReqRes](https://reqres.in/) | API REST para dados de usuários |
| [DiceBear](https://www.dicebear.com/) | API de geração de avatares |

---

## ⚙️ Como Funciona

Ao clicar no botão:

1. Um ID aleatório é gerado
2. Os dados são buscados na API
3. O nome do usuário é exibido
4. Um avatar é gerado automaticamente com base no nome

---

## 🎯 Objetivo do Projeto

Demonstrar habilidades fundamentais de front-end, como:

- Consumo de APIs
- Manipulação do DOM
- Programação assíncrona
- Tratamento de erros
- Boas práticas de UX

---

## 🚀 Como Executar

Basta abrir o arquivo `index.html` no navegador — sem dependências ou instalação necessária.
